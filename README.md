#D1 Number Guessing Game

This C++ program allows the user to guess a random number from 1 to 100. Each attempt indicates whether the number is too high or too low.

## How to Use

1. Open the file `D1_guessing the number.cpp` in your C++ compiler.

2. Compile and run the program.

3. Follow the instructions to guess the number.


### D2 - Password Strength Checker

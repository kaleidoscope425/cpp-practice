#D1 Number Guessing Game

This C++ program allows the user to guess a random number from 1 to 100. Each attempt indicates whether the number is too high or too low.

## How to Use

1. Open the file `D1_guessing the number.cpp` in your C++ compiler.

2. Compile and run the program.

3. Follow the instructions to guess the number.


### D2 - Password Strength Checker

- Uppercase letter
- Lowercase letter
- Number

Example:

Enter a password: Hello123

### D3 - Grade Analyzer

C++ program that calculates the average grade of 5 students, counts how many passed and failed, and validates input (grades must be between 0 and 5).

How to Use:
1. Open `D3_Grade Analyzer.cpp` in your C++ compiler.
2. Compile and run the program.
3. Enter grades when prompted.

### D4 - Calculator

This C++ program performs basic arithmetic operations between two numbers.

Features:
- Menu using switch
- Addition

### D5 - Sum of Digits

This C++ program asks the user for an integer and calculates the sum of its digits.

Example:

Input: 1234

### D6 - Prime Number Checker

This C++ program checks whether a number entered by the user is a prime number.

A prime number is only divisible by 1 and itself.

#### Example
Input: 7  
Output: The number is prime.
